## 🎬 Somino

Somino is a modern anime streaming and discovery platform built for Somali anime fans.
The platform provides anime information, trailers, and episode streaming in a clean, fast, scalable, and responsive environment.

## 🚀 About The Project

Somino is built using a modern, robust technology stack:
- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **State Management:** Zustand
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **PWA:** Fully installable via `@ducanh2912/next-pwa`
- **APIs:** Custom REST APIs (Anime info + multi-source streaming) & AniList GraphQL API

The goal of Somino is to create a private, high-quality, and deeply personalized anime platform focused on the Somali community.

## 🎯 Platform Goals

- High-performance, multi-server streaming
- Clean, modern, and glassmorphic UI design
- Deep community integration with leaderboards and profiles
- Seamless cross-device experience (PWA)
- Long-term scalability

## ✨ Features

- **🔎 Advanced Discovery & Filtering:** Browse by detailed genres, tags, season, and year. 
- **📺 Comprehensive Anime Details:** Synopsis, ratings, episodes, trailers, and character info.
- **🎞 Multi-Server Streaming:** High-quality video player supporting multiple servers (e.g. MegaPlay, VidWish) with user preferences for Autoplay, Auto-Next, and Auto-Skip.
- **🔐 User Authentication:** Secure Google and Email/Password login flows.
- **👤 User Profiles:** Personalized profiles with customizable settings, stats, and an AniList account synchronization feature.
- **📚 Watchlists & Favorites:** Track your Watch History, and save categorized Favorites (Anime, Manga, Characters).
- **📅 Schedule & Upcoming:** Stay updated with daily release schedules and upcoming anime seasons.
- **🏆 Leaderboards:** Community rankings and user leveling system.
- **📱 PWA Ready:** Install Somino as a native-like app on desktop and mobile devices.
- **⚡ Fast & Responsive:** Server-side rendering, optimized assets, and a fully responsive layout.

## 📂 Project Structure

- `app/` – Application routes and pages (App Router)
- `components/` – Reusable React UI components
- `lib/` & `utils/` – API handlers, configuration, and utility functions
- `types/` – TypeScript definitions
- `public/` – Static assets & PWA manifest
