## 🎬 Somino


Somino is a modern anime streaming and discovery platform built for Somali anime fans.
The platform provides anime information, trailers, and episode streaming in a clean, fast, and scalable environment.



## 🚀 About The Project:-
Somino is built using:
Next.js (App Router)
TypeScript
REST APIs (Anime info + streaming sources)
Modern responsive UI design
The goal of Somino is to create a private, high-quality anime platform focused on the Somali community.



## 🎯 Platform Goals

- High performance streaming
- Clean and modern UI
- Community-focused growth
- Long-term scalability



## ✨ Features

🔎 Anime search functionality
📺 Anime details page (synopsis, rating, episodes, trailer)
🎞 Episode streaming support
⚡ Fast performance with server-side rendering
📱 Fully responsive design
🔥 Scalable architecture for future growth




## 📂 Project Structure

app/ – Application routes and pages
components/ – Reusable UI components
lib/ – API handlers and utilities
types/ – TypeScript definitions
public/ – Static assets
